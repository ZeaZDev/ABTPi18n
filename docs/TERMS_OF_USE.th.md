# ข้อกำหนดและเงื่อนไขการใช้งาน Auto Bot Trader i18n (ABTPi18n)

**อัปเดตล่าสุด:** 10 พฤศจิกายน 2025
โปรดอ่านข้อกำหนดการใช้งานเหล่านี้ ("ข้อกำหนด", "ข้อกำหนดการใช้งาน") อย่างละเอียดก่อนใช้แพลตฟอร์ม Auto Bot Trader i18n ("บริการ") ที่ดำเนินการโดย ("เรา", "ของเรา")

การเข้าถึงและการใช้บริการของคุณขึ้นอยู่กับการยอมรับและปฏิบัติตามข้อกำหนดเหล่านี้ ข้อกำหนดเหล่านี้มีผลบังคับใช้กับผู้เยี่ยมชม, ผู้ใช้ และผู้อื่นทุกคนที่เข้าถึงหรือใช้บริการ

โดยการเข้าถึงหรือใช้บริการ คุณตกลงที่จะผูกพันตามข้อกำหนดเหล่านี้ หากคุณไม่เห็นด้วยกับส่วนใดส่วนหนึ่งของข้อกำหนด คุณจะไม่สามารถเข้าถึงบริการได้

## 1. เกี่ยวกับบริการ

Auto Bot Trader i18n (ABTPi18n) ให้บริการแพลตฟอร์มซอฟต์แวร์ที่ช่วยให้ผู้ใช้สามารถเชื่อมต่อกับบัญชี Exchange (ตลาดแลกเปลี่ยนคริปโต) ผ่าน API Keys และสั่งการกลยุทธ์การเทรดอัตโนมัติ ("บอท")

เราเป็นผู้ให้บริการเทคโนโลยี เรา **ไม่** ให้คำแนะนำทางการเงิน, คำแนะนำการลงทุน หรือบริการนายหน้าซื้อขายหลักทรัพย์

## 2. ความรับผิดชอบของผู้ใช้

คุณมีหน้าที่รับผิดชอบแต่เพียงผู้เดียวสำหรับ:
* **API Keys ของคุณ:** การรักษาความปลอดภัยบัญชี Exchange และ API Keys ของคุณ คุณต้องตั้งค่าสิทธิ์ที่เหมาะสม (เช่น เปิดใช้งานการเทรด, ปิดใช้งานการถอน) และใช้คุณสมบัติด้านความปลอดภัย เช่น IP Whitelisting ตามที่เราแนะนำ
* **กลยุทธ์ของคุณ:** การเลือก, การกำหนดค่า และการตรวจสอบกลยุทธ์การเทรดของคุณ คุณรับทราบว่าคุณใช้บอทเหล่านี้ตามดุลยพินิจของคุณเอง
* **ภาษีของคุณ:** การคำนวณและชำระภาษีใดๆ ที่อาจเกิดขึ้นจากกิจกรรมการเทรดของคุณ
* **การปฏิบัติตามกฎหมาย:** ตรวจสอบให้แน่ใจว่าการใช้บริการของคุณเป็นไปตามกฎหมายและข้อบังคับทั้งหมดในเขตอำนาจศาลของคุณ

## 3. ค่าธรรมเนียมและการสมัครสมาชิก

เราขอสงวนสิทธิ์ในการเรียกเก็บค่าธรรมเนียมหรือค่าสมัครสมาชิกสำหรับการใช้บริการ เราจะแจ้งให้คุณทราบล่วงหน้าอย่างเพียงพอเกี่ยวกับค่าธรรมเนียมดังกล่าว และคุณจะมีโอกาสยกเลิกบัญชีของคุณก่อนที่การเปลี่ยนแปลงดังกล่าวจะมีผลบังคับใช้

## 4. การใช้งานที่ต้องห้าม

คุณตกลงที่จะไม่ใช้บริการเพื่อวัตถุประสงค์ที่ผิดกฎหมายใด ๆ หรือเพื่อ:
* มีส่วนร่วมในการปั่นตลาด (เช่น โครงการ Pump and Dump)
* พยายามหลีกเลี่ยงหรือประนีประนอมกับคุณสมบัติด้านความปลอดภัยของเรา รวมถึงการเข้ารหัส
* ใช้บริการในลักษณะที่อาจทำให้ระบบทำงานหนักเกินไป, เสียหาย หรือบกพร่อง หรือรบกวนการใช้งานของบุคคลอื่น (เช่น เรียก API เกิน Rate Limits มากเกินไป)

## 5. การเปิดเผยความเสี่ยงที่สำคัญยิ่งยวด

การเทรดคริปโตเคอเรนซี, โดยเฉพาะอย่างยิ่งกับการใช้ Leverage (Futures) หรือระบบอัตโนมัติ, มีความเสี่ยงสูงและซับซ้อนอย่างยิ่ง โดยการใช้บริการนี้ คุณรับทราบและยอมรับความเสี่ยงดังต่อไปนี้:

* **ความเสี่ยงต่อการสูญเสียทางการเงิน:** คุณอาจสูญเสียเงินทุนบางส่วนหรือทั้งหมดของคุณ คุณไม่ควรเทรดด้วยเงินที่คุณไม่สามารถจะสูญเสียได้
* **ไม่รับประกันผลกำไร:** ผลการดำเนินงานในอดีตของกลยุทธ์ใด ๆ ไม่ได้บ่งชี้ถึงผลลัพธ์ในอนาคต ไม่มีการรับประกันว่าบอทใด ๆ จะทำงานตามที่คาดหวังหรือทำกำไรได้
* **ความเสี่ยงด้านตลาด:** มูลค่าของคริปโตเคอเรนซีมีความผันผวนสูง การร่วงลงอย่างรวดเร็ว (Flash Crash), การปั่นตลาด และการเปลี่ยนแปลงกฎระเบียบอย่างกะทันหันสามารถเกิดขึ้นได้
* **ความเสี่ยงจากการถูกบังคับปิดสถานะ (Liquidation):** หากเทรด Futures การเคลื่อนไหวของตลาดเพียงเล็กน้อยที่สวนทางกับสถานะของคุณอาจส่งผลให้สูญเสียมาร์จิ้นทั้งหมด (Liquidation)
* **ความเสี่ยงทางเทคนิค:** บริการนี้ต้องอาศัย API ของบุคคลที่สาม (เช่น Binance), เครือข่าย และซอฟต์แวร์ ความล้มเหลวในระบบเหล่านี้ (API ล่ม, การแบน IP จาก Rate Limit, บั๊กในบอท) อาจส่งผลให้เกิดการสูญเสียทางการเงิน และเราจะไม่รับผิดชอบต่อสิ่งเหล่านี้
* **ไม่ใช่คำแนะนำทางการเงิน:** ข้อมูล, กลยุทธ์ และเครื่องมือทั้งหมดที่ ABTPi18n ให้บริการมีไว้เพื่อจุดประสงค์ในการให้ข้อมูลเท่านั้น และไม่ถือเป็นคำแนะนำทางการเงินหรือการลงทุน

## 6. การปฏิเสธการรับประกันและการจำกัดความรับผิด

บริการนี้มีให้ "ตามที่เป็น" (AS IS) และ "ตามที่มี" (AS AVAILABLE) โดยไม่มีการรับประกันใด ๆ ไม่ว่าโดยชัดแจ้งหรือโดยนัย

ภายใต้ขอบเขตสูงสุดที่กฎหมายอนุญาต, ABTPi18n, บริษัทในเครือ, หรือผู้อนุญาต, ผู้ให้บริการ, พนักงาน, หรือตัวแทน จะไม่รับผิดชอบต่อความเสียหายใด ๆ (รวมถึงความเสียหายโดยตรง, โดยอ้อม, พิเศษ, โดยบังเอิญ, เป็นผลสืบเนื่อง หรือเชิงลงโทษ) ที่เกิดขึ้นจากหรือเกี่ยวข้องกับการใช้งานของคุณ หรือการไม่สามารถใช้งานบริการได้ ซึ่งรวมถึงแต่ไม่จำกัดเพียง การสูญเสียทางการเงิน, การสูญเสียข้อมูล, หรือความสูญเสียที่เกิดขึ้นเนื่องจาก:

* บั๊กหรือข้อผิดพลาดของบอทหรือซอฟต์แวร์
* การหยุดทำงานหรือข้อผิดพลาดของ API ของ EXCHANGE
* การถูกจำกัดการเรียก (RATE LIMITS) หรือการแบน IP จาก BINANCE (หรือ EXCHANGE อื่น)
* ข้อผิดพลาดในการตั้งค่ากลยุทธ์โดยตัวคุณเอง
* ความผันผวนของตลาดที่ไม่คาดคิด หรือเหตุการณ์ LIQUIDATION

## 7. ทรัพย์สินทางปัญญา

เราเป็นเจ้าของสิทธิ์, กรรมสิทธิ์ และผลประโยชน์ทั้งหมดในบริการ รวมถึงซอฟต์แวร์, โค้ด และอินเทอร์เฟซผู้ใช้ คุณไม่สามารถคัดลอก, แก้ไข หรือทำวิศวกรรมย้อนกลับ (reverse engineer) ส่วนใดส่วนหนึ่งของบริการของเราได้

## 8. การยุติบริการ

เราอาจยุติหรือระงับการเข้าถึงบริการของเราทันที โดยไม่ต้องแจ้งให้ทราบล่วงหน้าหรือรับผิด, ไม่ว่าด้วยเหตุผลใดก็ตาม รวมถึงแต่ไม่จำกัดเพียงหากคุณละเมิดข้อกำหนด

คุณสามารถยกเลิกบัญชีของคุณได้ตลอดเวลาโดยหยุดใช้บริการและเพิกถอน API keys ของคุณ

## 9. กฎหมายที่ใช้บังคับ

ข้อกำหนดเหล่านี้จะถูกควบคุมและตีความตามกฎหมายของ [Your Jurisdiction] โดยไม่คำนึงถึงข้อขัดแย้งของบทบัญญัติแห่งกฎหมาย

## 10. การเปลี่ยนแปลงข้อกำหนดเหล่านี้

เราขอสงวนสิทธิ์ตามดุลยพินิจของเราแต่เพียงผู้เดียวในการแก้ไขหรือแทนที่ข้อกำหนดเหล่านี้ได้ตลอดเวลา เราจะแจ้งให้ทราบถึงการเปลี่ยนแปลงที่เป็นสาระสำคัญ การที่คุณยังคงเข้าถึงหรือใช้บริการของเราต่อไปหลังจากที่การแก้ไขเหล่านั้นมีผลบังคับใช้ ถือว่าคุณตกลงที่จะผูกพันตามข้อกำหนดที่แก้ไขใหม่
